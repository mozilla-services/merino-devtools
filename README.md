## Mozilla Developer Tools

A collection of repositories to help us with our day-to-day work. Have in mind, that:

* Tools have to be vetted by the author and apply common security scrutiny
* Get in touch with the tool owner if you have any questions
* Tools might have had a single purpose, and are not up-to-date on the latest Merino codebase
* Some tools might be experiments. Treat them as such. Feel free to clone and modify freely to make use of them
